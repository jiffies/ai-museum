# AI入门指南

> 面向初学者的人工智能基础知识介绍

## 什么是人工智能？

人工智能（Artificial Intelligence，简称AI）是计算机科学的一个分支，致力于创建能够执行通常需要人类智能的任务的系统。

### AI的主要类型

1. **弱AI（Narrow AI）**：专注于特定任务的AI，如语音助手、推荐系统
2. **强AI（General AI）**：具备人类级别通用智能的AI（尚未实现）
3. **超级AI（Super AI）**：超越人类智能的AI（理论阶段）

## 机器学习基础

机器学习（Machine Learning）是AI的核心技术，让计算机能够从数据中学习。

### 学习类型

| 类型 | 描述 | 应用示例 |
|------|------|----------|
| 监督学习 | 使用标注数据训练 | 图像分类、垃圾邮件检测 |
| 无监督学习 | 从未标注数据发现模式 | 聚类、降维 |
| 强化学习 | 通过奖励信号学习 | 游戏AI、机器人控制 |

## 深度学习

深度学习使用多层神经网络处理复杂数据。

```
输入层 → 隐藏层1 → 隐藏层2 → ... → 输出层
```

### 常见架构

- **CNN（卷积神经网络）**：图像处理
- **RNN（循环神经网络）**：序列数据
- **Transformer**：自然语言处理

## 大语言模型（LLM）

大语言模型是当前AI领域最热门的技术，代表包括：

- GPT系列（OpenAI）
- Claude（Anthropic）
- Gemini（Google）

### LLM的能力

✅ 文本生成和理解
✅ 代码编写和调试
✅ 翻译和摘要
✅ 问答和对话
